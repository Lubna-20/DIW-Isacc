Ejercicio 1

Los siguientes campos numéricos representan la temperatura en grados centígrados y celsius. Con jQuery, consigue que cuando se modifique el valor de cualquiera de los inputs, el valor del otro se calcule automáticamente. Las fórmulas de conversión son: °F = (°C x 9/5) + 32 y °C = (°F - 32) x 5/9

<label for="celsius">Grados Celsius:</label>
<input type="number" id="celsius">
<br>
<label for="fahrenheit">Grados Fahrenheit:</label>
<input type="number" id="fahrenheit">

Ejercicio 3

Crea el código jQuery que transforme todas las vocales introducidas en un input de tipo texto por la letra 'i'.

Ejercicio 4

Crea el código jQuery para que cuando cualquier elemento de un formulario reciba el foco, todo el formulario obtenga un borde redondeado de color gris y una sombra. Cuando el formulario pierda el foco desaparecerá el borde y la sombra.

Ejercicio 5

Crea el código jQuery que compruebe la validez de un formulario de login. En el momento en que se pulse la tecla Enter en cualquiera de los inputs, se comprobará si ambos no están vacíos. Si alguno lo está, creará y mostrará dinámicamente un mensaje de aviso al usuario y pondrá un outline rojo al elemento vacío. Si cualquiera de los dos elementos tiene el outline cuando recibe el foco, este se elimina.

Ejercicio 6

Crea el código jQuery para que un input de tipo numérico solo pueda recibir números binarios, evitando la escritura de cualquier otro carácter distinto a '0' y '1'. Cuando se actualice el valor de ese input, aparecerá debajo el número correspondiente en decimal.

Ejercicio 7

Crea el código jQuery que compruebe si el valor introducido en un input de texto es una IPv4 válida.

Ejercicio 8

Crea el código jQuery que modifique el color del texto de un párrafo entre rojo, verde y azul a partir de 3 checkboxes.
