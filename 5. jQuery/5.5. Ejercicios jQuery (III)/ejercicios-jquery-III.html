°F = (°C x 9/5) + 32

Ejercicio 13

Crea el código jQuery que permita insertar alumnos en una lista y clonarlos a otra lista. La funcionalidad completa será la siguiente:

  1. Cuando el usuario pulse el botón #btnInsertar, se comprobará que tanto el nombre como los apellidos no están vacíos. Si alguno lo está, mostrará un mensaje de error en #mensaje. Si ambos no están vacíos, insertará en la lista izquierda al alumno ("Nombre" + "Apellidos").
  2. Si se pulsa sobre un elemento de la lista izquierda que no tenga la clase active, se le aplicará esta clase y se eliminará de los demás elementos.
  3. Si se pulsa sobre un elemento con la clase active, esta clase se eliminará.
  4. Al pulsar el botón "Clonar", se clonará el elemento con la clase active de la lista izquierda a la lista derecha. El elemento de la lista izquierda quedará marcado con un atributo data-clonable que evitará que pueda ser activado o clonado de nuevo.
  5. Se mostrará un mensaje tanto si se ha clonado un elemento como si no.
  6. Todos los mensajes en #mensaje irán con el estilo correspondiente en alerts de Bootstrap (danger para mensajes de error, success para mensajes de éxito).

HTML:

<div class="container-selection">
  <div class="list-container">
    <h4>Lista de personas</h4>
    <ul id="personasList">
      <li>Ana García Pérez</li>
      <li>Javier Martínez López</li>
    </ul>
  </div>
  
  <div class="btnContainer">
    <button id="btnClonar">Clonar →</button>
  </div>
  
  <div class="list-container">
    <h4>Elementos clonados</h4>
    <ul id="listaClonados">
      <!-- Aquí se añadirán los elementos clonados -->
    </ul>
  </div>
</div>

CSS:

.container-selection {
  display: flex;
  max-width: 800px;
  margin: 0 auto;
}

.list-container {
  flex: 1;
  border: 1px solid #ccc;
  padding: 15px;
  margin: 0 10px;
  border-radius: 5px;
  min-height: 200px;
}

.btnContainer {
  display: flex;
  flex-direction: column;
  justify-content: center;
  padding: 10px;
}

#btnClonar {
  padding: 10px 15px;
  background-color: #4CAF50;
  color: white;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}
  
#btnClonar:hover {
  background-color: #45a049;
}

ul {
  list-style-type: none;
  padding: 0;
}

li {

li:hover {
