Ejercicio 19

Crea el código jQuery para realizar una petición AJAX al cargar la página al archivo JSON json/ej-19.json que contiene información sobre coches. La aplicación debe funcionar así:

  1. Al cargar la página, se realizará una petición AJAX para obtener los datos del archivo JSON.
  2. Con los datos recibidos, se extraerán las marcas de coches únicas y se generará dinámicamente un checkbox con su correspondiente etiqueta para cada marca en el formulario proporcionado.
  3. Si el archivo JSON está vacío o no contiene datos, se mostrará un mensaje de error en el área de resultados.
  4. El botón "Ver coches" debe permanecer deshabilitado hasta que se reciban correctamente los datos de la primera petición AJAX.
  5. Cuando el usuario seleccione una o varias marcas y pulse el botón "Ver coches", se realizará una nueva petición AJAX al mismo archivo JSON. Se filtrarán los coches según las marcas seleccionadas por el usuario y se mostrarán en el área de resultados, indicando para cada coche: marca, modelo y precio.
  6. Si no se ha seleccionado ninguna marca al pulsar el botón, se mostrará un mensaje informativo.
